# Apenas 1 vez
git config --global user.name -> Adiciona o nome de Usuário do GitHub
git config --global user.email -> Adiciona o Email de usuário do GitHub
git init -> inicializa um repositórtio local
git add remote [nomeDaBranch] "url-do-github"
-------------------------------------------------------
git status -> Verifica o estado que o respositório local se encontra
git pull -> atualiza o repositórtio local e deixa igual ao repositório remoto
git add . -> adiciona no respositório local todos os arquivos que estão na pasta 
git commit -m "Alguma coisa" -> Comento quais funcionalidades foram adicionadas
git push -> adiciona os arquivos no github
git branch nomedabranch-> Cria uma nova branch
git checkout nomedabranch -> mudar de branch
